# Admin Maintenance Mode

## Overview

Superadmins can toggle the application into maintenance mode from the admin panel. Uses Laravel's built-in `artisan down`/`up` commands with auto-generated bypass secrets.

## How It Works

- **Cache-based config** — Maintenance state, custom message, and bypass secret are stored in the application cache (`maintenance_mode` key).
- **Artisan integration** — Toggling on calls `artisan down --with-secret`, toggling off calls `artisan up`.
- **Bypass secret** — A unique URL is generated that allows specific users to bypass the maintenance page.

## Routes

| Method | URI | Action |
|--------|-----|--------|
| GET | `/admin/maintenance` | Display maintenance settings |
| POST | `/admin/maintenance/toggle` | Toggle maintenance mode |

## Frontend

Admin page at `resources/js/Pages/admin/maintenance.tsx` provides:

- Status badge (live/maintenance)
- Warning banner when maintenance is active
- Bypass secret URL with copy button
- Custom message field
- Toggle button that changes based on current state

## Authorization

Only users with `is_superadmin = true` can access the maintenance routes (enforced by `superadmin` middleware).
